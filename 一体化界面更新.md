## 深度思考功能

### 什么是深度思考？

豆包的深度思考模式会让AI更仔细地分析题目，给出更准确的答案。

### 何时使用？

- ✅ **推荐使用**：难题、复杂题、重要考试
- ❌ **不推荐**：简单题、练习题、时间紧张时

### 区别对比

| 模式 | 速度 | 准确率 | 适用场景 |
|------|------|--------|----------|
| 普通模式 | 快 | 高 | 一般题目 |
| 深度思考 | 慢 | 更高 | 难题、重要考试 |

## 技术实现

### API服务自动启动

程序使用 `subprocess` 启动API服务：

```python
self.api_process = subprocess.Popen(
    [sys.executable, "app.py"],
    stdout=subprocess.PIPE,
    stderr=subprocess.PIPE
)
```

### 深度思考参数

在调用豆包API时添加参数：

```python
payload = {
    "prompt": "...",
    "use_deep_think": True  # 启用深度思考
}
```

## 注意事项

1. **关闭程序时** - API服务会自动停止
2. **深度思考模式** - 会比普通模式慢一些
3. **API状态** - 确保显示"运行中 ✓"再开始答题

## 常见问题

### Q: API服务启动失败？
A: 检查8000端口是否被占用，或查看日志了解详情

### Q: 深度思考有多慢？
A: 大约比普通模式慢2-3倍，但准确率更高

### Q: 可以中途切换深度思考吗？
A: 可以！停止答题后重新勾选即可
