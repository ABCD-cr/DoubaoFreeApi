# 自动答题系统 - 快速开始

## 5分钟快速上手

### 第一步：安装依赖

双击运行 `安装依赖.bat` 或执行：
```bash
pip install -r requirements.txt
```

### 第二步：配置Session（重要！）

**必须先配置 `session.json` 文件才能使用系统。**

参考 `如何配置Session.md` 文件获取并配置你的豆包账号信息。

### 第三步：启动系统

双击运行 `启动自动答题.bat` 或执行：
```bash
python auto_answer_main.py
```

### 第四步：启动API服务

在打开的界面中：
1. 点击"启动API服务"按钮
2. 等待显示"API: 运行中 ✓"

### 第五步：配置系统

1. **选择题目区域**
   - 点击"选择题目区域"按钮
   - 用鼠标框选题目显示的区域
   - 释放鼠标完成

2. **配置答案选项**
   - 点击"配置答案选项"按钮
   - 依次点击A、B、C、D选项的位置
   - 最后点击"下一题"按钮的位置

### 第六步：开始答题

1. 输入题目总数（如：10）
2. （可选）勾选"启用深度思考"以提高准确率
3. 点击"开始答题"
4. 系统会自动完成所有题目！

## 注意事项

✅ **必须先配置 session.json 文件**
✅ 题目必须是A/B/C/D格式的选择题
✅ 答题过程中不要移动答题界面
✅ 第一次使用建议先测试1-2题
✅ **使用图片识别模式，无需安装OCR软件**
✅ 深度思考模式更准确但更慢

## 技术优势

🎉 **一体化界面**
- API服务和答题系统集成在一个窗口
- 无需开两个终端窗口
- 操作更简单直观

🎉 **直接发送图片给豆包AI识别**
- 不需要安装Tesseract-OCR
- 识别更准确（豆包的多模态能力）
- 配置更简单

🎉 **深度思考选项**
- 对于难题可以启用深度思考
- 提高答题准确率

## 遇到问题？

**Q: 提示"会话配置不存在"？**
A: 需要先配置 `session.json` 文件，参考 `如何配置Session.md`

**Q: API服务启动失败？**
A: 检查端口8000是否被占用，或查看日志文件

**Q: 识别不准确？**
A: 尝试启用"深度思考"选项，或调整题目区域选择范围

查看详细文档：[README_AUTO_ANSWER.md](README_AUTO_ANSWER.md)
